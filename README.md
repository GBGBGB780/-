# 数字百团

#### 介绍
中山大学软件工程学院21级项目实训——“数字百团”项目，数字百团项目是一个旨在管理和协调校园内社团活动的综合性平台。该平台涵盖了校团委、各社团及申请者三个主要用户端，提供从社团信息管理到年检审批、申请处理等一系列功能。通过数字化手段，旨在提高社团管理的效率、透明度和便捷性，同时为学生提供更加丰富和便利的社团参与体验。
